# Route each model to its own backend
$ corvia inference reload \
  --model nomic-embed-text-v1.5 --backend openvino

$ corvia inference reload \
  --model llama3.2 --backend cuda

Each model gets its own backend. All Rust. All local. No API keys.

Takeaways
What I Learned
This Week
⚡
Your iGPU is free compute.
Split workloads instead of fighting for one GPU.
🔌
ONNX sessions are independent.
One on CUDA, one on OpenVINO, one on CPU. All parallel.
🛠
Detect at startup, not at build time.
One binary. Probe hardware. Pick backends. Fall back gracefully.
