The Pattern
Same Mistakes.
Every Time.
After dozens of iterations, the same four
categories of visual errors kept appearing.
📄

Text Overflow

Long labels spill out of fixed-width containers.
📌

Element Overlap

Absolute positioning without checking neighbors.
📈

Misaligned Diagrams

Flow arrows that don't connect. Boxes at different heights.
🚲

Grid Collapse

4-column grid collapses to one. Side by side becomes stacked.
The frustrating part: fix one issue, ask for another change, and a different alignment breaks.

The Fix
Give AI Eyes.
Playwright MCP.
Playwright MCP lets Claude Code launch a browser,
render the page, and screenshot what it built.
✍ Write HTML/CSS
→
🌐 Render page
→
📸 See screenshot
↓
👀 Spot overflow
→
🔧 Fix the CSS
→
✅ Verify fix
// .mcp.json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["@playwright/
        mcp@latest"]
    }
  }
}
Same AI, same code generation. Now it can see what it produces and fix it — just like a human would.

The Difference
Context Is
Everything.
The model didn't change. The tools around it did.
🚫

Without Playwright

• Write code → hope it works
• Fix one thing → break another
• 3–5 rounds of back-and-forth
• Human becomes the "eyes"
• Tedious and frustrating
✅

With Playwright

• Write → screenshot → verify
• Spots own mistakes first
• Self-correcting loop
• Human reviews final result
• Faster and autonomous
This applies beyond UI
PDFs
Slides, reports, guides
Dashboards
React, data viz, admin
Emails
HTML across clients

The model is rarely the bottleneck. The context is.
Give AI the right tools to close the feedback loop,
and it goes from frustrating to genuinely useful.

Key Takeaways
What I Learned
Building UI with AI
👁
AI can code UI. It can't see UI.
Valid CSS doesn't mean correct layout.
The rendering gap is real.
📄
Alignment is the failure mode.
Text overflow, element overlap, misalignment
— the same issues every time.
🔧
Tools beat prompts.
No amount of "be careful with alignment"
in a prompt fixes this. A screenshot helps.
💡
Context engineering > prompt engineering.
Don't ask AI to be better. Give it the right
tools to close the loop.
